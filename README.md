# She Can Foundation - Frontend Internship Task
A modern and responsive NGO landing page created for the She Can Foundation Frontend Development Internship Task.
## Overview
This project is a professional single-page website designed for She Can Foundation. The website highlights the mission of empowering women through education, leadership, skill development, and community support.
The design focuses on creating a clean, modern, and user-friendly experience.
## Features
- Responsive Navigation Bar
- Hero Section with Background Image
- About Section
- Impact Statistics Cards
- Hover Effects and Animations
- Modern NGO Landing Page Design
- Mobile Responsive Layout
## Technologies Used
- HTML5
- CSS3
